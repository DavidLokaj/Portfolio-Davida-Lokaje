Úvod O mně Úkoly Finální projekt

O tomto webu

Tento web prezentuje moji práci a pokrok v rámci projektu ZPC 2025. Najdete zde jednotlivé úkoly, mé osobní cíle a závěrečný projekt.

Popis úkolu

Prvním úkolem do předmětu ZPC bylo vytvoření vlastních stránek, postup byl následující:

  1. Instalace potřebného softwaru (Hugo, Git, Visual Studio Code).
  2. Vytvoření nového projektu.
  3. Přidání šablony a základního obsahu.
  4. Zprovoznění lokální verze.
  5. Nahrání na GitHub Pages pomocí Actions.

Jak začít

Začněte v sekci Úvod, kde najdete první kroky mé cesty. Poté si můžete projít Úkoly, které popisují jednotlivé etapy mé práce.
